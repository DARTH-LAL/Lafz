Lafz Library Tools

into Lafz.

Playlist Import

Add a full playlist to your queue.

Paste a Spotify playlist URL or ID. Lafz fetches every track, deduplicates them, and adds them to your translation queue instantly.

⚠ Spotify's Development Mode may block public playlists you don't own. If you see a 403 Forbidden error, copy the playlist to your own library first.
Single Song Import

Add one track without a playlist.

Paste a Spotify track URL or ID. Lafz pulls the song metadata and adds it directly to your queue — no playlist needed.

What happens when you import

Instant, local, private.

Everything stays on your machine. No data leaves Lafz except the Spotify API call to fetch track metadata.

🎵
Tracks are fetched from Spotify
Lafz reads the playlist or track metadata using your connected Spotify account.
🗂
Duplicates are merged automatically
Songs that appear in multiple playlists are deduplicated by Spotify track ID.
📝
Translation files are created
A local translation stub is written for every new track, ready for lyric and AI work.
✨
Queue is ready immediately
Head to the Library Queue to see all imported songs and start translating.
Only playlists you own or collaborate on can be imported
Existing translation files are never overwritten
Single-song imports automatically fetch lyrics if available
